For quick setup:
----------------

1. Download and import the Motion Controller
https://www.assetstore.unity3d.com/en/#!/content/15672

2. Download and import the Spell Casting Motion Pack

3. Download Mixamo’s Animations per the User's Guide 
https://www.mixamo.com

4.Unzip animations to project’s Mixamo folder
Assets\ootii\MotionControllerPacks\SpellCasting\Content\Animations\Mixamo

5. Unzip animation metadata to project’s Mixamo folder
Assets\ootii\MotionControllerPacks\SpellCasting\Extras
Assets\ootii\MotionControllerPacks\SpellCasting\Content\Animations\Mixamo

6. Close and re-open Unity

7. Open the demo scene

8. Setup input entries (on MC and on Packs tab)

9. Press play
